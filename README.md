# small_pro
A simple user management system

## env 
* Laravel 5.7 
* node >= 8.9.0
* yarn >= 1.10.0
* php 7.2
## init 
1. composer install
	* cp .env.example .env
	* php artisan key:generate
	php artisan admin:install
2. yarn install
3. php artisan migrate
4. php artisan db:seed 
5. npm run dev 

`if env is production       
please exec `npm run pro` `
